JS 类

1.数据类型：

console.log('0' == false);   // true 隐式转换相等，都为0
console.log('0' === false);  // false  数据类型不同
console.log(0 == false);    // true  隐式转换相等，都是0
console.log(0 === false);    // false  数据类型不同
console.log(null == undefined);  // true  undefined值是派生自null值的
console.log(null === undefined);  // false 数据类型不同

总结：

  • 数据类型的种类：

    （1）数值型：Number、String、Boolean、Null、Undefined

    （2）引用型或复杂性：Object、Array

  • js中 === 和 == 的区别：

    （1）== 用于比较判断两者相等，会自动转化数据类型，只要求值相等

    （2）=== 用于严格比较，判断两者严格相等，要求值和类型都相等
